# 心音检测说明

## 关于心音智鉴

### 产品简介

心音智鉴是一款基于人工智能的心音检测应用，通过树莓派边缘设备采集心音数据，结合深度学习模型进行实时分析，为用户提供便捷的心脏健康自检服务。

### 核心功能

| 功能 | 说明 |
|------|------|
| 心音采集 | 通过电子听诊器采集30秒心音数据 |
| AI分析 | 边缘AI模型实时分析心音特征 |
| 风险评估 | 提供三级风险评估（正常/中等/高风险） |
| 健康追踪 | 云端存储检测记录，追踪健康趋势 |
| AI助手 | 智能健康问答和个性化建议 |

## 检测前准备

### 设备准备

1. 确保树莓派检测设备已开机
2. 检测设备与手机连接至同一WiFi网络
3. 通过小程序扫码或手动输入IP连接设备

### 个人准备

为获得最佳检测效果，请注意：

- **休息5分钟**：检测前请静坐休息，让心率稳定
- **避免剧烈运动**：检测前30分钟内避免剧烈运动
- **穿着宽松**：便于放置听诊器探头
- **保持安静**：检测环境尽量安静，减少噪音干扰
- **避免说话**：检测过程中请勿说话或移动

## 检测步骤详解

### 第1步：连接设备

1. 打开心音智鉴小程序
2. 点击"扫码连接"扫描设备二维码
3. 或点击"手动输入"输入设备IP地址
4. 等待连接成功提示

### 第2步：准备检测

1. 点击"开始检测"按钮
2. 按照屏幕指引，将听诊器探头放置在心脏位置
3. **最佳位置**：左胸第4-5肋间，锁骨中线内侧
4. 确保探头与皮肤贴合良好

### 第3步：采集心音

1. 系统开始30秒心音采集
2. 屏幕显示实时心音波形
3. 保持静止，正常呼吸
4. 避免说话和大幅度移动
5. 倒计时结束后采集完成

### 第4步：AI分析

1. 系统自动进行AI心音分析
2. 分析过程约需3-5秒
3. 分析完成后显示检测结果

### 第5步：查看结果

结果页面显示：
- 风险等级评估
- 各类别概率分布
- 个性化健康建议
- 就医建议（如需要）

## 检测结果解读

### 风险等级说明

| 等级 | 颜色 | 含义 | 建议 |
|------|------|------|------|
| 正常 | 🟢 绿色 | 心音特征正常 | 继续保持健康生活方式 |
| 中等风险 | 🟡 黄色 | 检测到轻微异常 | 建议近期就医检查 |
| 高风险 | 🔴 红色 | 检测到明显异常 | 建议尽快就医 |

### 分析类别

AI模型分析以下心音类别：
- **Normal**：正常心音
- **Murmur**：心脏杂音
- **Extrasystole**：心律不齐/早搏
- **Artifact**：噪音干扰

### 置信度说明

- 置信度表示AI模型对判断结果的确定程度
- 置信度越高，结果越可信
- 若置信度较低，建议重新检测

## 常见问题

### Q1：检测结果显示"异常"怎么办？

首先不要过度紧张。心音智鉴检测结果仅供参考，不能作为诊断依据。建议：
1. 在安静环境下重新检测一次
2. 如果多次检测均显示异常，建议就医
3. 将检测记录提供给医生参考

### Q2：为什么同一天检测结果不同？

心音可能受多种因素影响：
- 检测时的姿势和呼吸状态
- 检测前的活动情况
- 探头放置位置的微小差异
- 环境噪音干扰

建议在相似条件下进行检测，以便对比。

### Q3：检测需要多久进行一次？

- 一般建议每周1-2次常规检测
- 身体不适时可随时检测
- 检测频率可根据个人情况调整

### Q4：儿童可以使用吗？

可以，但请注意：
- 儿童心率正常范围与成人不同
- 检测时需保持儿童安静
- 结果解读需考虑年龄因素
- 如有疑问请咨询儿科医生

## 使用注意事项

### 重要声明

1. 心音智鉴是健康自检工具，**不是医疗设备**
2. 检测结果**仅供参考**，不能替代医学诊断
3. 任何健康问题请**及时就医**
4. 紧急情况请立即**拨打120**

### 不适用人群

以下情况可能影响检测准确性：
- 有心脏起搏器
- 胸部有手术疤痕
- 皮肤过敏或损伤
- 极度肥胖

### 设备维护

- 使用后清洁听诊器探头
- 定期检查设备连接线
- 保持设备干燥清洁
- 避免强烈撞击
